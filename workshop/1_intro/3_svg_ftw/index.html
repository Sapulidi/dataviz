Op een gegeven moment wordt er gevraagd om een CSV bestand, deze staat al klaar in de map waar dit index bestand ook in staat. Je hoeft dus niets meer te downloaden. Wat je wel moet doen is dat voorbeeld in een webserver omgeving draaien omdat er een extern bestand wordt ingeladen. Mensen op een mac kunnen eenvoudig een server in een map starten door het commando 'python -m SimpleHTTPServer' in een terminal op te geven. Voor mensen met een windows machine is mongoose een makkelijke oplossing.

Om te testen kunnen jullie hiernaast altijd gebruik maken van oege.ie.hva.nl (de nieuwe stap). Jullie kunnen daarop een account aanmaken.

Een andere oplossing is in Chrome toestaan lokale bestanden in te laden via JavaScript, de meest eenvoudige maar ook de meest gevaarlijke oplossing.

  • Mac: open een terminal en type: 'open -a Google\ Chrome --args -–allow-file-access-from-files'
  • Win: open een command prompt, cd naar de map waar Chrome.exe staat en type: 'chrome.exe -–allow-file-access-from-files'
